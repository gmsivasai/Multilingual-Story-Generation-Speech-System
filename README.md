# 🌍 AI Multilingual Story Generation & Speech System

An AI-powered multilingual storytelling web application developed using Django, GPT-2, NLP, and Text-to-Speech technologies.  
The system generates creative stories from user-provided keywords, translates them into multiple languages, and converts them into speech audio.

---

## 🚀 Project Overview

This project combines Generative AI, Natural Language Processing (NLP), Machine Translation, and Speech Synthesis into a single interactive web application.

Users can:
- Generate AI-based stories from keywords
- Translate stories into multiple languages
- Listen to generated stories using speech synthesis
- Interact through a simple web interface

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/gmsivasai/Multilingual-Story-Generation-Speech-System.git
```

---

### 2️⃣ Navigate to Project Folder

```bash
cd Multilingual-Story-Generation-Speech-System
```

---

### 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

---

### 4️⃣ Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/Mac

```bash
source venv/bin/activate
```

---

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 6️⃣ Run Django Server

```bash
python manage.py runserver
```

---

### 7️⃣ Open Browser

```bash
http://127.0.0.1:8000/
```

## 🎮 How It Works

1. User enters story keywords
2. GPT-2 generates a story
3. Story is translated into selected language
4. gTTS converts text into speech
5. User can read or listen to generated story
